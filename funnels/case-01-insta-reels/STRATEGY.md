## 1. 퍼널 설계 의도 (Conversion Intent)

본 퍼널은 단순히 템플릿을 홍보하는 것이 아니라, **시청자의 체류 시간(Watch Time)을 극대화**하고 **자발적 참여(Engagement)**를 끌어내어 인스타그램 알고리즘과 노션 유입을 동시에 잡는 것이 핵심입니다.

### 🎯 핵심 메커니즘: "정보의 비대칭성과 보상"
- 영상에서 가치를 70%만 명시적으로 보여주고, 나머지 30%의 '실행법(템플릿)'을 보상으로 제시하여 행동을 촉발합니다.

## 2. 단계별 마케팅 트리거 (Marketing Triggers)

### 🪝 PHASE 1. 3초 후킹 (The Visual Hook)
*목표: 스크롤을 멈추게 하는 강력한 시각적/언어적 충격*
- **시각적 트리거**: 'Before & After'의 드라마틱한 대비 또는 '속도감 있는 화면 전환'.
- **언어적 트리거**: 
  - "이거 모르면 인스타 헛하는 겁니다." (손실 회피)
  - "퇴근 1시간 앞당겨준 유일한 도구" (이익 제공)
  - "나만 알고 싶은 노션 치트키 공개" (희소성)

### 📺 PHASE 2. 시청 유지 (The Retention)
*목표: 이탈 없이 영상 끝까지 보기 + 알고리즘 점수 확보*
- **편집 전략**: 텍스트 자막을 0.5초~1초 단위로 빠르게 배치하여 시청자의 눈을 바쁘게 만듦.
- **음향 전략**: 비트감이 강하거나 트렌디한 오디오 사용.
- **내용 구성**: 궁금증을 유발하는 시연 영상을 보여주되, "세부 설정법은 댓글 확인"이라는 자막을 중간에 아주 짧게 노출(재시청 유도).

### 💬 PHASE 3. 댓글 액션 (The Trap: Comment to DM)
*목표: 댓글 알고리즘 활성화 및 개인별 맞춤 리드 생성*
- **전환 트리거**: "댓글로 `신청`이라고 적으시면 제가 직접 만든 템플릿과 매뉴얼을 DM으로 보내드립니다."
- **심리적 장치**: "선착순 ○○명에게만 공개" 또는 "마감 임박" 키워드로 긴박감 조성.
- **자동화 연동**: 매니챗(ManyChat)이 실시간으로 확인 답글을 달아줌으로써 게시물 활성도를 폭발시킴.

### 🔗 PHASE 4. 노션 리드 유입 (The Conversion)
*목표: 인스타 유입자를 노션 도메인으로 안착시켜 잠재적 리드로 전환*
- **DM 랜딩**: "축하합니다! 방금 신청하신 템플릿 링크입니다."
- **노션 안착**: 단순히 파일만 주는 것이 아니라, **"이 템플릿을 제대로 쓰는 법"** 가이드 문서를 노션 최상단에 배치하여 신뢰도 확보.

## 3. 알고리즘 점수화 전략 (Score Strategy)

| 행동 | 가중치 | 대응 전략 |
| :--- | :--- | :--- |
| **댓글(Comment)** | ★★★★★ | "신청" 키워드 유도 및 1분 내 답글 자동화 |
| **공유(Share)** | ★★★★☆ | "주변에 필요한 친구에게 태그" 유도 |
| **저장(Save)** | ★★★★☆ | "나중에 꺼내 보려면 저장 필수" 문구 삽입 |
| **재시청(Re-watch)** | ★★★☆☆ | 루프(Loop) 형태의 영상 편집 또는 정보성 자막 빠르게 돌리기 |
